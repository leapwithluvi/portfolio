## рџљЂ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 18
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/leapwithluvi/portfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies

```

```bash

```


